# Portfólio de Análise de Dados com Python

Bem-vindo ao meu portfólio de projetos Python para Análise de Dados! Este repositório complementa meu [portfólio SQL](https://github.com/Matheus-Diniz-Silva/portfolio-sql-analise-dados) demonstrando habilidades em Python para manipulação, visualização e análise de dados.

## Sobre

Sou um Analista de Dados em formação e aqui documento minha jornada de aprendizado e aplicação prática do Python para análise de dados. Este portfólio apresenta projetos progressivos que mostram desde manipulação básica de dados até análises estatísticas avançadas.

## Habilidades que serão Demonstradas

### Manipulação de Dados
- Pandas para limpeza e transformação de dados
- NumPy para cálculos numéricos
- Trabalho com diferentes formatos (CSV, JSON, Excel)

### Visualização de Dados
- Matplotlib para gráficos básicos e customizados
- Seaborn para visualizações estatísticas
- Plotly para gráficos interativos

### Análise Estatística
- Estatística descritiva e exploratória
- Testes de hipóteses
- Análise de correlações e tendências

### Ferramentas e Boas Práticas
- Jupyter Notebooks para análise exploratória
- Scripts Python reproduzíveis
- Versionamento com Git e GitHub
- Documentação clara e objetiva

## Projetos Planejados

### [01 - Análise Exploratória com Pandas](/01-analise-exploratoria-pandas)

### [03 - Análise Estatística](/03-analise-estatistica)
